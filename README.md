# Notepad App

A simple notes app built using Node.js, Express, MongoDB, and EJS.

## Live Demo
https://notepad-app-hveu.onrender.com

## Features
- Create notes
- Edit notes
- Delete notes
- Pin notes

## Tech Used
- Node.js
- Express.js
- MongoDB
- EJS
